Program tests the led lights by executing commands turn on, turn off, and switch and returns number of LEDs on.
Program takes as an input url to a txt file.
Commands in text file must be written line by line in the following format:

command rowFrom,columnFrom through rowTo,columnTo

Valid commands are:
switch, turn on, turn off

row and column numbers have to be positive integers
=============================================
=============================================
|| To run the program from command line use:||
|| led_tester --input "url to the txt file" ||
